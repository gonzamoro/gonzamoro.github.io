App de contactos


Proyecto de progressive web app que desarrollé para el lugar donde trabajo, la cual guarda datos de proveedores y clientes.


Este proyecto fue ideado para reemplazar la agenda física escrita a mano, por una agenda virtual en la cual se puedan agregar editar y borrar contactos cuando se necesite.
